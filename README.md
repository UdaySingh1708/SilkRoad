# Silk Road Anonymous Chat

Anonymous real-time stranger chat platform built with Node.js and Socket.IO.

## Features

- Real-time anonymous messaging
- Random stranger matching
- Online user counter
- Typing indicator
- Next stranger button
- Disconnect system
- Rate limiting
- Security headers
- Responsive design

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express
- Socket.IO
